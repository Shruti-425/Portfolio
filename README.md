###  My  Portfolio

A modern, responsive developer portfolio built using **React, Vite, Tailwind CSS, and Framer Motion**, showcasing projects, technical expertise, and contact information.

**Live Website:**
 [https://shruti-425.github.io/Portfolio/](https://shruti-425.github.io/Portfolio/)

---

##  Features

* Responsive design (mobile + desktop)
* Smooth animations using Framer Motion
* Clean UI with Tailwind CSS
* Resume download functionality
* Deployed using GitHub Pages
* Optimized production build with Vite

---

##  Tech Stack

* **Frontend:** React, Vite
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Deployment:** GitHub Pages
* **Version Control:** Git & GitHub

##  Installation & Setup

Navigate to the project:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Deploy to GitHub Pages:

```bash
npm run deploy
```
